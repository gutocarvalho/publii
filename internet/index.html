Internet

Conectividade no Via

Atualmente só temos esses dois provedores atendendo nosso prédio:

- Skillnet Telecom
- R2 Telecom

Ambos provedores oferecem internet via fibra ótica até o modem.

Ambos provedores oferecem planos com boa velocidade de download, sendo equivalentes na disponibilidade do serviço.

Ambos provedores oferecem IP Fixo, o qual pode ser contratado a parte.

Sobre Upload
- A skill.net oferece upload equivalente a 50% da velocidade de download contratada
- A R2 oferece upload equivalente a 25% da velocidade de download contratada

Caso sua necessidade seja maior Upload, contrate a Skill.net.

Multi-WAN + Balanceamento de Links

Alguns moradores – como eu – tem os dois provedores para garantir que possam trabalhar sem interrupção em homeoffice.

Para que isso ocorra de forma transparente, sem mexer com cabos ou trocar de rede é preciso de um roteador multi-wan com balanceamento de links e também de um sistema wifi independente, de preferência com tecnologia MESH.

Segue a recomendação de equipamentos multi-wan-lb residenciais:

- https://www.tp-link.com/br/business-networking/vpn-router/er605/  router/lb
- https://www.tp-link.com/br/home-networking/wifi-router/deco-m5/ antena wifi mesh
- https://www.tp-link.com/br/business-networking/easy-smart-switch/tl-sg108e/ switch

Equipamento de nível profissional:

- https://www.ui.com/unifi-routing/usg/ router/lb
- https://br.store.ui.com/products/unifi-ac-mesh-ap antena wifi mesh
- https://br.store.ui.com/products/unifi-switch-8-60w switch

Não se esqueça de desligar o WIFI dos modens e usar só do sistema independente.

Aqui no 901-A eu já usei:

- Roteador TP-Link R600VPN
- Access Point TP-Link Deco M5 ( 2 unidades )

Hoje uso:

- Unifi USG como roteador para 2 internets
- Unifi Switch8 60 para ligar todos os dispositivos e levar cabos para alguns quartos
- Unifi Antena UAP-AC-M-BR Mesh A/C ( 2 unidades )

Desliguei o WIFI nos roteadores da Skill e R2 e tudo tem funcionado muito bem, nem percebo quando a R2 ou SKILL saem do ar, sigo navegando de forma transparente.
